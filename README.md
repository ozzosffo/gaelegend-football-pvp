# 개레전드 축구 PVP

브라우저에서 바로 접속해 즐기는 실시간 멀티플레이 축구 프로토타입입니다. GitHub Pages에서 정적 사이트로 구동되며, 멀티플레이는 PeerJS/WebRTC 기반 P2P로 동작합니다. 한 명의 브라우저가 방 호스트가 되어 경기 상태를 계산하고, 다른 플레이어는 그 호스트에 연결됩니다.

## GitHub Pages 배포

이 저장소를 GitHub에 올리고 `main` 브랜치에 push하면 `.github/workflows/pages.yml` 워크플로가 `public/` 폴더를 GitHub Pages로 배포합니다.

GitHub 저장소 설정에서 Pages source가 `GitHub Actions`로 되어 있어야 합니다.

## 로컬 확인

`public/index.html`을 브라우저로 열어 화면을 확인할 수 있습니다. 실제 온라인 멀티플레이는 GitHub Pages처럼 HTTPS로 열린 주소에서 가장 안정적입니다.

방은 기본적으로 6자리 숫자 코드로 만들어집니다. 링크 복사를 누르면 `https://사용자명.github.io/저장소명/?room=123456` 같은 숫자 링크가 복사되고, 그 주소를 직접 열면 같은 방 코드가 자동으로 입력됩니다.

## 멀티플레이 방식

- `자동`: 열린 방이 있으면 참가하고, 없으면 내 브라우저가 호스트가 됩니다.
- `HOME`: 내 브라우저가 방 호스트가 됩니다.
- `AWAY`: 이미 열린 방에 참가합니다.

PeerJS의 무료 Cloud signaling 서버를 사용합니다. 게임 데이터는 중앙 서버가 아니라 WebRTC 데이터 채널로 브라우저끼리 주고받습니다.

## 기본 조작

- 방향키: 이동
- `E`: 전력 질주
- `C`: 볼 지키기/견제
- `S`: 짧은 패스, 수비 중 압박/견제
- `W`: 스루 패스
- `A`: 로빙 패스/크로스, 수비 중 슬라이딩 태클
- `D`: 슛, 수비 중 태클/걷어내기
- `Z + D`: 감아차기
- `Q`: 팀원 침투 요청/수비 지원 요청

FC온라인의 대표 키 배치를 브라우저용으로 옮긴 오리지널 게임입니다.
